Projetos

1. Configuracao de Rede com VLANs e Inter-VLAN Routing

Ferramentas: Cisco Packet Tracer

Objectivo: Separar departamentos em redes distintas melhorando segurança e performance.

Topologia:

  • VLAN 10 - Administracao
  • VLAN 20 - Vendas
  • VLAN 30 - TI
  • Router-on-a-stick

Comandos principais:

enable; conf t; vlan 10; vlan 20; vlan 30; interface g0/1; swictport mode trunk; interface g0/1.10; encapsulation dot1q 10; ip adress 192..168.10.1 255.255.255.0;

Resultado:

Dispositivos de redes diferentes comunicando via router-on-a-stick. Ping bem-sucedido entre VLANs.

2. Servidor DHCP no Windows Server 2022

Objetivo: Automatizar atribuiçao de enderenço IP para 50 Maquinas.

Passos:

  • Instalei o DHCP Role
  • Criei um escopo 192.168.1.10-192.168.1.200
  • Configurei reservas para impressoras e cameras
  • Testes com ipconfig/renew

Resultado:

A rede ficou totalmente automatica e sem conflitos IP.
